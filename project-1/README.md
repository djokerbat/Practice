# Drink Website 🍹

This project is a multi-page beverage website UI designed in Figma and developed using HTML and CSS. It includes three different product pages for beverages with a consistent and responsive layout.


📌 Project Overview

The goal of this project was to convert a Figma design into a functional and responsive website using only HTML and CSS, focusing on UI accuracy and clean structure.
This folder contains the design and the code of the project.

## 🛠️ Technologies Used
- HTML5
- CSS3
- Figma (for UI design)
