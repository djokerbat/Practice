# NIKE Sneakers Website 👟


## 📌 Project Overview

This project is a multi-page sneakers website UI designed in Figma and developed using HTML and CSS. Each page represents a different Nike sneaker model with a unique color theme while maintaining a consistent and responsive layout

  
## 🛠️ Technologies Used

* Figma for UI design

* Multi-page layout for different sneaker models
* Unique color theme for each product page
* Responsive design for different screen sizes
* Clean and organized UI structure
